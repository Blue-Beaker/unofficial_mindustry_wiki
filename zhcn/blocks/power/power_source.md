# 无限能源
[![无限能源](/images/block-power-source-xlarge.png)无限能源](power_source.md)  

类似[能量节点](/zhcn/blocks/power/power_node.md),但自带能源产出,且可建立100个连接.  
截至126版本,每个无限能源的发电量固定为1M,不受超速影响.

### 属性

| 属性 | 数值 |  
| ---- | ---- |  
|ID|power-source|
|生命值|40|  
|尺寸|1x1|
|建造时间|0秒|
|建造花费|仅无限资源|
|连接范围|6|
|连接数|100|
|阻挡地面单位|是|
|发光|仅能量激光|
|可超速|否|
|enabled参数|控制是否产生能量|  
|config参数|无|
